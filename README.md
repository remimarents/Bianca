# Biancas Tetris (Web)

Dette er en webversjon av Biancas Tetris, laget for å kunne publiseres på GitHub Pages.

## Hvordan bruke

1. Last opp innholdet i `web_tetris`-mappen til et nytt GitHub-repo.
2. Aktiver GitHub Pages under repoets "Settings" → "Pages" og velg `main`-branch og `/ (root)` som kilde.
3. Spillet blir tilgjengelig på `https://<brukernavn>.github.io/<reponavn>/`.

## Videre arbeid
- Spillmotoren i `tetris.js` må ferdigstilles for å matche funksjonaliteten i Python-versjonen.
- Highscore kan lagres i LocalStorage.
- Lyd og grafikk kan forbedres etter behov.

Se `tetris_bianca.py` for original logikk og funksjonalitet.
